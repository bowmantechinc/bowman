MSSP Deploy
Connect to Google Sheets as your free cloud database — one Apps Script deployment hosts both the data and this page, so it can be embedded directly in Google Sites.
1
Go to sheets.google.com and create a new spreadsheet. Name it MSSP_PM_DB.
2
In the spreadsheet, go to Extensions → Apps Script. Delete the default code in Code.gs and paste the Apps Script below (click Show Apps Script code).
3
In the Apps Script editor, click File → New → HTML file, name it exactly Index (no extension), and paste the entire contents of this same mssp_app.html file into it.
4
Click Deploy → New deployment → Web app. Set Execute as: Me and Who has access: Anyone. Copy the Web App URL ending in /exec — paste it below to test the connection.
5
To embed in Google Sites: open your Site → Insert → Embed → By URL → paste the same /exec URL. That's it — one URL serves the whole app and stays connected to this Sheet.
